{% trans "About OpenSrc Persian" %}

{% blocktrans %}OpenSrc Persian is a free, community-driven platform with a single purpose: making knowledge available in Persian and sharing it with the world, without any barriers. We believe that language should never be a reason someone is excluded from learning — and that access to knowledge is a right, not a privilege.{% endblocktrans %}

  • {% trans "Book translations" %}
  • {% trans "Game subtitle localizations" %}
  • {% trans "Tutorials, guides, and educational resources" %}
  • {% trans "Discussions around free software and digital rights" %}

{% blocktrans %}Everything on this site is completely free. No subscriptions, no payments, no advertisements — and no hidden agenda. (Maybe.){% endblocktrans %}

{% trans "How We Work" %}

{% blocktrans %}OpenSrc Persian is run entirely by volunteers. We are not able to pay contributors, and we say so honestly. What we offer instead is a space where your work reaches people who genuinely need it — and where it stands alongside others who share the same values. If you are passionate about spreading knowledge, translating useful content, promoting scientific thinking, or simply contributing to the Persian-speaking community — we invite you to join us.{% endblocktrans %}

{% trans "What We Stand For" %}

{% blocktrans %}Beyond content, we care about the principles behind it. OpenSrc Persian supports organizations that defend software freedom and an open internet. We encourage our community to write about free and open-source software issues and share their knowledge, exchange their experiences, and engage in the broader global conversation about digital rights. An informed community is a stronger one.{% endblocktrans %}
